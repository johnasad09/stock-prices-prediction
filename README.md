## 🎯 Task Objective
The goal of this task is to predict the next day’s closing stock price using historical market data. The focus is on applying machine learning models to understand patterns in stock movements 📈.

## 📊 Dataset Used
Historical stock data is used, collected from Yahoo Finance using the yfinance library. The dataset includes features such as Open, High, Low, Close, and Volume.

## 🛠️ Libraries Used
- pandas → for data handling and preprocessing 
- matplotlib & seaborn → for data visualization 📉  
- yfinance → for fetching stock market data  
- scikit-learn → for building and training models 🤖  

## 🤖 Models Applied
- Linear Regression → used as a baseline model for predicting stock prices  
- Random Forest Regressor → used to capture non-linear relationships and improve prediction performance 🌳  

## 🔍 Key Results and Findings
- Historical stock data reveals useful patterns for short-term price prediction.  
- Features like Open, High, Low, and Volume contribute to predicting the Close price.  
- Linear Regression provides a simple and interpretable baseline.  
- Random Forest delivers better performance by handling complex patterns in the data.  
- Predictions generally follow market trends, but accuracy varies due to market volatility ⚠️.  
- Using multiple models provides better insights and more robust predictions 🚀.
